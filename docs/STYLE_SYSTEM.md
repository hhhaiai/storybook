# 📖 绘本风格标准化系统 (Style System)

## 概述

基于 Google Gemini 参考风格，建立了一套可复用的儿童绘本内容生产标准。

核心定位：**"儿童职业启蒙 + 安全感教育 + 正向价值观"**

---

## 一、故事风格系统 (Narrative System)

### 🎯 核心定位
- 目标受众：3-8岁儿童
- 情绪曲线：🙂平静 → 😟轻微紧张 → 😌放松 → 😊温暖+崇拜
- 禁止内容：暴力细节、恐惧氛围、复杂人性冲突

### 📐 统一5段式结构

```
1. 场景引入（轻松、安全）
2. 问题出现（小冲突，不制造恐惧）
3. 警察/角色介入（专业 + 温和）
4. 问题解决（正向结果）
5. 情感收束（信任 / 成长 / 梦想）
```

### 📚 故事模板库（7个主题）

| ID | 名称 | 主题 | 默认角色 |
|---|---|---|---|
| `lost_child` | 迷路救援 | 安全教育 | 警察+男孩+妈妈 |
| `traffic_safety` | 交通安全 | 社会规则 | 警察+男孩 |
| `fire_rescue` | 火灾救援 | 帮助他人 | 消防员+男孩 |
| `detective_mystery` | 刑警破案 | 勇敢与责任 | 刑警+男孩 |

| `armed_rescue` | 武警救援 | 帮助他人 | 武警+男孩 |
| `dream_job` | 职业认知 | 梦想与职业 | 警察+男孩 |

### 🗣️ 语言风格
- 短句（8-18字）
- 口语化
- 高可读性
- 示例："小朋友，不用怕，我在这里。"

## 二、图片风格系统 (Visual System)

### 🎨 视觉风格

#### 推荐风格：软扁平绘本 (Soft Flat)

```
核心特征：
- 儿童绘本 (storybook)
- 软扁平 (soft flat)
- 低光影 (low shading)
- 高亲和力 (friendly)
- 情绪导向 (emotion-first)
```

#### 色彩体系

```
主色调：
- 绿色（安全 / 自然）
- 蓝色（警察 / 信任）
- 米色（温暖）
- 珊瑚色点缀

特点：
- 低对比
- 高明度
- 无强烈阴影
- 柔和渐变
```

#### 构图规则

```
- 中景（人物占60%）
- 轻微仰视（增强"守护感"）
- 互动构图（牵手 / 对视）
- 干净背景
```

### 🎨 可用视觉风格

| ID | 名称 | 描述 |
|---|---|---|
| `soft_flat` | 软扁平绘本（推荐） | Gemini参考风格 |
| `soft_flat_warm` | 暖色软扁平 | 金色阳光感 |
| `soft_flat_cool` | 冷色软扁平 | 清新宁静感 |

## 六、Prompt 组装流程

### 图片生成

```
1. 视觉风格（锁死风格） → BasePrompt
2. 色彩体系 → ColorScheme
3. 场景检测 → MapSceneFromText()
4. 角色描述 → Character.Description
5. 动作描述 → Action
6. 情绪检测 → MapEmotionFromText()
7. 构图规则 → Composition
8. 光影规则 → Shading
9. 无文字约束（最强）
```

### 故事生成

```
1. 核心定位（职业启蒙+安全感+正向价值观）
2. 模板选择（detectTemplateID）
3. 5段式结构
4. 情绪曲线
5. 强约束（纯中文、无英文）
6. 单页规则（1-3句、8-18字）
7. 语言风格（短句口语化）
8. 表达覆盖（情绪词+动作词）
9. 收尾要求（温暖+安全感）
```

## 七、使用方式

### Web 控制台

1. 启动服务：`./storybook --serve`
2. 访问：`http://localhost:8080`
3. 选择"🎨 软扁平绘本（推荐）"画风
4. 选择故事风格标签
5. 点击生成

### 命令行

```bash
./storybook "小明的警察梦"
```

## 九、风格一致性保障

### 防止风格漂移

1. **视觉风格锁死**：所有图片 prompt 最前面放 BasePrompt
2. **角色一致性**：每个角色有固定英文描述，全文档不变
3. **色彩体系固定**：pastel 色板写入每个 prompt
4. **构图规则统一**：中景+轻微仰视+互动构图
5. **无文字约束**：最强约束放在 prompt 最后

### 场景/情绪自动检测

- `MapSceneFromText()`：根据中文文本关键词自动选择场景
- `MapEmotionFromText()`：根据中文文本关键词自动选择情绪
- 无需手动指定，系统自动匹配
