# AI Storybook Web Generator

## 项目状态：✅ 标准化风格系统已集成

---

## 一、项目目标

```
项目名称：AI Storybook Web Generator

目标：
构建一个本地可运行的 Web 应用，实现：

1. 一键生成儿童绘本（20页）
2. 类似 Gemini 的阅读体验（封面 + 左图右文）
3. 支持翻书动画（真实翻页）
4. 支持导出：
   - PDF（通过浏览器打印）
   - 图片（每页PNG/JPG）
5. 支持分享（生成静态HTML，图片base64嵌入）
6. 标准化风格系统（防漂移、角色一致性）
```

## 三、核心功能

### 1️⃣ 绘本生成

```
输入：
- 用户输入一句话（主题）

输出：
- 20页绘本（文本 + 图片）

规则：
- 每页 1 张图 + 1 段中文文字
- 自动调用：
  - 文本API（通过 .env 配置）
  - 图片API（通过 .env 配置）
- 自动失败重试（最多5次）
- 纯色图片检测 + 自动重试
```

### 2️⃣ 标准化风格系统

```
核心定位：儿童职业启蒙 + 安全感教育 + 正向价值观

包含：
- 8个标准角色（警察/武警/特警/刑警/消防员/男孩/女孩/妈妈）
- 3种视觉风格（软扁平推荐/暖色/冷色）
- 6个标准场景（公园/街道/学校/家/树林/警察局）
- 8种情绪映射（平静/担心/害怕/安心/开心/崇拜/坚定/关爱）
- 7个故事模板（迷路救援/交通安全/火灾救援/刑警破案/特警出击/武警救援/职业认知）
- 自动场景检测（MapSceneFromText）
- 自动情绪检测（MapEmotionFromText）
```

### 3️⃣ 页面结构

#### 📕 封面页（Page 0）
```
元素：
- 大图（封面图，风格系统生成）
- 标题（居中）

样式：
- 图片全屏
- 半透明遮罩
- 标题居中（大字体）
```

#### 📖 内容页（Page 1-20）
```
布局：
| 左侧：图片（64%） | 右侧：文字（36%） |

要求：
- 图片占满左侧
- 文字垂直居中
- 留白充足（像绘本）

文字规则：
- 21px
- 行高1.9
- 首字下沉（drop-cap）
```

### 4️⃣ 翻书效果

```
实现：纯 CSS 3D transform + JavaScript

支持：
- 点击左右翻页
- 拖拽翻页
- 键盘 ← →

动画：
- 翻页阴影
- 页面弯曲（3D rotateY）
```

### 5️⃣ 导出功能

```
PDF：浏览器打印功能（window.print()）
图片：逐页下载原始 JPG
```

## 四、配置管理

### 环境变量（唯一账号源）

所有账号配置集中在 `.env`：

```
# API 配置（必填）
USER_BASE_URL=https://your-api-endpoint.com/v1
USER_API_KEY=your-api-key-here

# 模型配置
TEXT_MODEL=grok-4.20-0309-non-reasoning
IMAGE_MODEL=grok-imagine-image-lite

# 生成配置
MAX_RETRIES=5
REQUEST_TIMEOUT=120
IMAGE_DOWNLOAD_TIMEOUT=50
PORT=8080
```

### 启动校验

缺失 `USER_API_KEY` 或 `USER_BASE_URL` 时，服务启动报错退出。

## 五、使用方法

### 编译
```bash
./build.sh
```

### 运行
```bash
# 启动 Web 服务
./storybook --serve

# 访问控制台
open http://localhost:8080
```

### 画风选择

控制台默认选中"🎨 软扁平绘本（推荐）"，共 20 种画风可选。
