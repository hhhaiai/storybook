# 📚 AI Storybook Generator（本地绘本生成器）

基于 Grok / OpenAI 兼容 API 的儿童绘本自动生成工具，支持：

- ✅ 自动生成 20 页中文故事（可调 12-30 页）
- ✅ 自动生成插画（软扁平 / 水彩 / 蜡笔等 20 种风格）
- ✅ 标准化风格系统（角色一致性 / 场景检测 / 情绪映射）
- ✅ Web 控制台管理（选择风格、模型、主题）
- ✅ 本地 HTML 翻页阅读器
- ✅ PDF 导出

---

## 🚀 功能特性

- 封面海报页（标题 + 插画）
- 左图右文（标准绘本结构）
- 8 个标准角色（警察 / 武警 / 特警 / 刑警 / 消防员 / 男孩 / 女孩 / 妈妈）
- 7 个故事模板（迷路救援 / 交通安全 / 火灾救援 / 刑警破案 / 特警出击 / 武警救援 / 职业认知）
- 自动重试图片生成 + 纯色检测
- 批量生成支持
- 本地运行，无需外部前端框架

---

## 📦 安装

需要 Go 1.21+：

```bash
# 编译
./build.sh

# 或手动编译
go build -o storybook ./cmd/server
```

---

## ⚙️ 配置

复制 `.env.example` 为 `.env` 并填入 API 密钥：

```bash
cp .env.example .env
```

编辑 `.env`：

```bash
# API 配置（必填）
USER_BASE_URL=https://your-api-endpoint.com/v1
USER_API_KEY=your-api-key-here

# 模型配置
TEXT_MODEL=grok-4.20-0309-non-reasoning
IMAGE_MODEL=grok-imagine-image-lite

# 服务配置
PORT=8080
```

支持 OpenAI 兼容接口的任何模型（Grok / GPT-4o / Claude / DeepSeek / 通义千问等）。

---

## ▶️ 运行

```bash
./storybook
```

访问控制台：http://localhost:8080

---

## 📂 输出

每次生成会在 `outputs/` 下创建独立目录：

```
outputs/
└── story-20260529-211824/
    ├── data.json       # 绘本元数据 + 每页文本
    ├── index.html      # 翻页阅读器（直接打开）
    └── images/         # 每页插画
        ├── 1.jpg       # 封面
        ├── 2.jpg
        └── ...
```

## 📖 阅读绘本

在控制台点击故事卡片，或直接打开：

```bash
open outputs/story-20260529-211824/index.html
```

支持点击翻页、拖拽翻页、键盘 ← → 翻页。

## 🧠 生成逻辑

1. 规划故事名称与主题（LLM）
2. 翻译角色描述为英文（用于图片 prompt）
3. 生成故事文本（中文）
4. 为每页生成插画描述（英文）
5. 生成插画（自动重试 + 纯色检测）
6. 渲染 HTML 绘本

```
├── outputs/                 # 生成的故事输出
├── build.sh                 # 编译脚本

```
